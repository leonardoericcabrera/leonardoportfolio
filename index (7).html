🎮 Enterprise TMS Implementation — FrostStorm Games
Case Study · Middlebury Institute of International Studies · 2025–2026
TMS Evaluation
Led a team of three localization specialists to evaluate and recommend an enterprise-grade Translation Management System for a fictional live-service game studio. The project simulated a real-world procurement process: identifying stakeholder requirements, designing a controlled pilot test, and producing a data-driven final recommendation for executive buy-in.
Methodology
Controlled pilot using a 300-word JSON file + 10-term lore termbase to test Phrase vs memoQ Enterprise
Stakeholders Mapped
7 key stakeholders including IT, Localization PMs, Audio Team, Security, and Finance
Recommendation
memoQ Enterprise (score 4.55 vs 3.95) — superior IP protection, terminology management & LQA hard-stops
Key Insight
Transitioning QA from a reactive bottleneck to a proactive, automated gatekeeper via built-in LQA validation
